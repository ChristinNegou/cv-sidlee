Projet phare

Deneige Auto

Full-Stack Mobile

Plateforme mobile connectant clients et déneigeurs en temps réel.

Problème

Les étudiants et propriétaires perdent du temps à chercher un service de déneigement fiable.

Solution

Une plateforme mobile complète avec paiement intégré et communication temps réel.

Backend Node.js / Express / MongoDB
  • API REST sécurisée avec authentification JWT
  • Hash des mots de passe (bcrypt)
  • Paiements intégrés avec Stripe
  • WebSockets (Socket.IO) — temps réel
  • Upload d'images (Multer + Cloudinary)
  • Sécurisation API (Helmet, Rate limiting)
  • Tests automatisés (Jest, Supertest)
  • Déploiement cloud (Railway + Docker)
Frontend Mobile Flutter
  • Clean Architecture (domain / data / presentation)
  • State management Bloc
  • Intégration Google Maps + géolocalisation
  • Notifications push (Firebase)
  • Gestion sécurisée des tokens
